
# Spring Boot Restaurant API

A RESTful API for managing restaurants and their dishes, built with **Java**, **Spring Boot**, and **Spring Data JPA**.  
This project is part of my learning journey in building real-world backend services with proper layering, validation, and error handling.

- Uses **MySQL** (or H2) as the database
- Uses **JPA/Hibernate** for ORM
- Basic validation and exception handling
- Designed to be easily extendable (add users, orders, reviews, etc.)
